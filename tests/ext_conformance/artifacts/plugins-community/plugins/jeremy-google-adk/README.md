# Google Agent Development Kit (ADK) Plugin

Google ADK SDK starter kit for building Claude-powered AI agents with React patterns, multi-agent orchestration, and tool integration.

## Features

- Agent scaffolding and boilerplate generation
- React-style component patterns for agent development
- Multi-agent orchestration support
- Tool integration helpers
- Best practices for Google ADK development

## Installation

```
/plugin install jeremy-google-adk@claude-code-plugins-plus
```

## Usage

Once installed, the plugin provides skills and commands for working with Google ADK projects.
