# Vertex AI Integration Plugin

Comprehensive Vertex AI integration for building generative AI agents with Gemini, Vertex AI Studio, and production deployment on Google Cloud.

## Features

- Gemini model integration and prompt engineering
- Vertex AI Studio workflows
- RAG (Retrieval-Augmented Generation) patterns
- Vector search with embeddings
- MLOps best practices for LLM deployment
- Cloud Run deployment templates
- Agent Builder scaffolding

## Installation

```
/plugin install jeremy-vertex-ai@claude-code-plugins-plus
```

## Usage

Once installed, the plugin provides skills and commands for working with Vertex AI and Gemini on Google Cloud Platform.
